# dm-chain-utils

Shared Python utilities for the `dd_chain_explorer` streaming applications.

Published to PyPI as `dm-chain-utils`.

## Installation

```bash
pip install dm-chain-utils
```

## Modules

- `dm_chain_utils` — Web3 helpers, AWS SDK wrappers (Kinesis, SQS, DynamoDB, Firehose), and shared constants used across the Docker streaming jobs.
